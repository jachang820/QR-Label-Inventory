# Smokebuddy

Smokebuddy inventory system.

### Installing

Run the following

```
npm install
```

## Built With

* [Node.js 10.14.2](https://nodejs.org/) - Javascript runtime
* [Express.js 4.16.0](https://expressjs.com/) - The web framework used
